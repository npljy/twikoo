## 部署步骤（草稿）

1. 申请 [MongoDB](https://www.mongodb.com/cloud/atlas/register) 和 [Vercel](https://vercel.com/signup) 账号
2. 创建免费 MongoDB 数据库，区域推荐选择 `AWS / N. Virginia (us-east-1)`
3. 创建免费 Vercel 项目，区域推荐选择 `Washington, D.C., USA (East) – iad1`
